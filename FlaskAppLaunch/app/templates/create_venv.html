How to Create a Virtual Environment from your Command Terminal

In your command terminal type in the following code:

virtualenv venv

Notes: "venv" is the name you are giving your virtual environment. You don't have to name it "venv", you can name it whatever you like, though "venv" is commonly used in the coding community. Also, before submitting the command to activate the virtual environment, be sure that you are located in the main directory for the app.
