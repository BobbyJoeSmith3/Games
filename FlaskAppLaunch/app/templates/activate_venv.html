How to Activate and Deactivate a Virtual Environment from your Command Terminal

In your command terminal type in either of the following lines of code:

source venv/bin/activate

-or-

. venv/bin/activate

Notes: before submitting the command to activate the virtual environment, be sure that you are located in the main directory for the app.In both of these instances, "venv" is the name of the virtual environment you are activating. "venv" is commonly used in the coding community, but you could name it whatever you like. You will know that the virtual environment has been activtated because you will see at the beginning of your command line in parenthesis the name you gave the virtual environment ex. (venv)yourcomputerid@suchandsuch.com:$
